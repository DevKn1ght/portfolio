Projet Laravel / PHP

Lors de notre apprentissage de PHP, nous avions du apprendre à utiliser et produire un site web avec le framework Laravel. Le contexte du site web est de pouvoir s'inscrire en tant qu'utilisateur, pour voir et réserver des denrées/donations. Le but fut de nous évaluer sur l'apprentissage et l'utilisation de Laravel, ainsi que l'implémentation d'une base de données comportant des liens relationnels.
